# -Criando-Um-Relat-rio-Gerencial-de-Vendas-com-Power-BI

  Este projeto tem como objetivo desenvolver um dashboard gerencial de vendas utilizando o Power BI, com foco na análise de dados e apoio à tomada de decisões estratégicas. O relatório foi construído a partir de uma base de dados de vendas (sample financials), permitindo a visualização clara de indicadores importantes como faturamento, lucro e desempenho por segmento.
